Descrevendo algumas TAGS HTML*

Tag < em >

Esta Tag é utilizada para dar ênfase ao texto. Por padrão o texto fica em Itálico.

Tag < strong >

Esta tag é utilizada para dar importância ao texto. Por padrão o texto fica em Negrito.

Tag < sup > e < sub >

Estas Tags são utilizadas para definir um texto sobreescrito e sobreescrito H20 22=4

Tag < ins > e < del >

Estas Tags são utilizadas para definir inserções e remoções de texto. Esse tipo de conteúdo é muito comum em blogs. Texto a ser Deletado Texto a ser inserido

Tag < mark >

Esta Tag é utilizada para marcar trechos do texto como se fosse uma caneta marca texto. parte do texto a ser marcada
*HTML : Hyper Text Markup Language
